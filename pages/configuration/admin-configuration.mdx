# Admin Configuration

Admin configuration is to control the transferability of shares and loot.

```js
    function setAdminConfig(bool pauseShares, bool pauseLoot)
        external
        baalOrAdminOnly
    {
        sharesPaused = pauseShares; 
        lootPaused = pauseLoot; 
        emit SharesPaused(pauseShares);
        emit LootPaused(pauseLoot);
    }
```
